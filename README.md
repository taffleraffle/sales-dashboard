# OPT Digital — Sales Dashboard

Standalone sales analytics and pipeline management dashboard for OPT Digital.

## Overview

This application provides:
- **GHL Pipeline Analytics** — Funnel tracking, stage distribution, auto-booked vs manual setter comparison
- **Wavv Dialer Metrics** — Pickup rates, meaningful conversations, calls-to-set ratios per stage
- **Speed to Lead** — Response time tracking from lead creation to first action
- **EOD Reports** — Daily closer and setter performance submissions
- **Lead Attribution** — Setter → Closer → Outcome tracking with commission calculations
- **Marketing Tracker** — Ad spend, CPA, ROAS, and full-funnel daily metrics
